Light Able Vue

Open source Vue admin template

Getting Started

This section explains how to install and run Light Able Vue for the first time. It covers the prerequisites and commands needed to start development.

To get started with this project, you will need to have the following installed:

  • Node.js v14.0.0 or higher
  • npm v6.0.0 or higher | yarn 1.22.21 or higher
  • Vue CLI

Install dependencies:

npm install or yarn add

Install dependencies

npm install or yarn install

Install Vue CLI

npm install -g @vue/cli | yarn add -g @vue/cli

Compiles and hot-reloads for development

npm run serve | yarn serve

Compiles and minifies for production

npm run build | yarn build

Lints and fixes files

npm run lint | yarn lint

Folder Structure

Folder Use-Description
public contains static assets like images, HTML files
src contains Vue components, routers, store etc.
components reusable Vue components
tests unit and end-to-end tests
readme.md readme file
node_modules dependencies
package.json metadata
.gitignore git config
assets static assets
logo.png logo image
app.vue main app component
main.js entry point
router.js router config
store.js state management
example.spec.js test file

HTML Structure

The main App component is in App.vue. This contains the router-view component which renders the different page components based on the route.

Each page component can be found in the pages folder, e.g. Home.vue, About.vue.
